# PlaywrightTS-Practice
Practicing [**Playwright Framework**](https://playwright.dev/) for **Web GUI** and **APIs** Test Automation.

## Setup:
1. Install [**Node.jsÂ®**](https://nodejs.org/en/download/)
2. Install [**VS Code IDE**](https://code.visualstudio.com/download) and install **Playwright Test for VSCode** plugin and you can install **Material Icon Theme** for better icon visibility.
3. Clone the project in a new VS Code folder.
4. Install **Playwright Framework** with the following commands in order: ```npm install playwright --save-dev``` & ```npm install @playwright/test --save-dev```.
5. Finally, install **Playwright Browsers**: ```npx playwright install```.

## Execution:
* To execute all the test cases please run the following command from the terminal/commandline: ```npx playwright test```.
* You can also run from the GitHub Actions workflow.
